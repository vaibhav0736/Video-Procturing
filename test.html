🎯 Video Proctoring System

Advanced AI-powered video monitoring for online interviews and exams

✅ Project setup completed successfully!

🚀 Key Features Implemented

🎥 Real-time Video Monitoring
Live candidate video feed with AI-powered analysis
👤 Advanced Face Detection
MediaPipe integration for accurate face tracking and gaze analysis
🔍 Focus Detection
Monitors if candidate looks away >5s or no face detected >10s
📱 Object Detection
TensorFlow.js COCO-SSD detects phones, books, and suspicious items
📹 Video Recording
Records and saves interview sessions locally
📊 Event Logging
Comprehensive real-time logging with timestamps and alerts
